# CS Coursework

A collection of coursework and exercises from university classes,
organized by topic.

## Contents

### Object-Oriented Programming (`oop/`)
- `rectangle_class.cpp` - Class with area and perimeter methods
- `pointers_basics.cpp` - Pointer declaration, arithmetic and dereferencing
- `personnel_class.cpp` - Multiple objects with private fields and setter
- `house_class.cpp` - Encapsulation with getters, setters and validation

### Inheritance (`inheritance/`)
- `inheritance_products.cpp` - Base class with derived classes using constructors and method overriding

### Database (`sql/`)
- `customer_queries.sql` - SELECT, WHERE, LIKE, JOIN, GROUP BY and subqueries

## Languages
C++ · SQL
